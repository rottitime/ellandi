# Ellandi (Front-End)

## First-time setup

```sh
git clone https://github.com/tbjgolden/sensible-starter.git project-name
cd project-name
npm install
git init                          # reinits git repo
npm run prepare                   # installs git hooks
git add .
git commit -m 'Initial commit'
```

## Usage

For all commands run:

```sh
npm run help
```

To run in dev mode:

```sh
npm run dev
```

To run in prod mode:

```sh
npm run prod
```
